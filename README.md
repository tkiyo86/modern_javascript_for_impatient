# javascript 5つの原則
- 変数はvarでなく、letまたはconstで宣言せよ
- strictモードを使え
- 型を理解し、自動的な型変換を避けよ
- プロトタイプを理解しよう。しかし、クラスとコンストラクタとメソッドには現在の構文を使おう。
- thisは、コンストラクタまたはメソッドの中だけ使おう
